# 戦略設計レポート：農業従事者向けマッチングアプリ

**作成日：** 2026年6月28日  
**分析手法：** PEST分析 → SWOT分析 → 3C分析

---

## 1. PEST分析

### Political（政治・法規制）

| 要因 | 内容 | 事業への影響 |
|------|------|-------------|
| スマート農業技術活用促進法（2024年施行） | 農業DXを国が推進。農業マッチングプラットフォームも支援対象になり得る | ✅ 補助金・税制優遇の活用機会 |
| 食料・農業・農村基本法改正（2024年〜） | 農業人材確保・担い手育成が国の重点施策に格上げ | ✅ 政策追い風。行政連携の窓口が広がる |
| 省力化投資促進プラン（農林水産省、2025年6月） | 農業の省力化・デジタル化投資を国が後押し | ✅ 農家のデジタルサービス導入コストが低減 |
| 地域計画の全国義務化（2026年3月まで） | 全市町村が農業人材ニーズを計画化 → 行政連携でニーズ情報取得が容易に | ✅ 行政との農家マッチング連携機会 |
| 特定技能「農業」区分の拡大 | 外国人就労者の農業従事が合法化・増加中 | ✅ 差別化機能として多言語マッチング需要が拡大 |
| 農家のデータ管理・個人情報規制 | マッチング履歴・給与情報の取扱いに個人情報保護法が適用 | ⚠️ コンプライアンスコストが発生 |

### Economic（経済）

| 要因 | 内容 | 事業への影響 |
|------|------|-------------|
| 農業生産コストの高騰 | 肥料・燃料・資材費の上昇で農業所得が圧迫。食料自給率37%（カロリーベース） | ⚠️ 農家の採用予算が縮小する可能性 |
| 副業市場の拡大 | 企業・自治体が副業を相次いで解禁。スポット就労者の供給が構造的に増加 | ✅ ワーカー側の供給量が増加し、マッチング成立率が上がる |
| 人材紹介業界の成長 | 2024年度市場規模4,490億円（前年比+12%）。デジタルマッチング移行が全業種で加速 | ✅ 投資家からの評価・調達しやすい市場環境 |
| スマート農業市場の拡大 | 農業用ドローン市場2025年1,073億円、農業ロボット665億円に成長 | ✅ 農業テック全体への投資・注目が追い風 |
| 農村部の経済活性化ニーズ | 地方移住・ワーケーション推進を国が政策化 | ✅ 都市部ワーカーの農業参加に補助金がつく可能性 |

### Social（社会・文化）

| 要因 | 内容 | 事業への影響 |
|------|------|-------------|
| 農業就業者の急減・高齢化 | 就業者180万人（2024年）、平均年齢69.2歳、65歳以上が約70%。年間7万人減 | ✅ 人手不足の深刻化でマッチング需要が増大 |
| Z世代の農業関心の高まり | Z世代の約65%が農業に関心。短期就労・ボランティア希望率は64.1%で全世代平均（36.8%）を大幅上回る | ✅ ワーカー側の供給候補層が豊富 |
| 地方移住・田舎暮らし志向 | 移住相談件数61,720件（2024年）、4年連続過去最高。東京圏15〜29歳の45.6%が地方暮らしに憧れ | ✅ 農業体験ニーズと地方移住ニーズが重なるユーザー層を取り込める |
| 副業意識の定着 | 副業を本業の収入補完として当然視する意識が若年層を中心に定着 | ✅ 「週末農業バイト」の需要が底堅い |
| 高齢農家のITリテラシー問題 | スマホ・アプリ操作に抵抗感を持つ65歳以上農家が多く、既存サービスの活用率が低い | ⚠️ 農家側の獲得コストが高くなりやすい |
| 食・農業への価値観の変化 | 食の安全・地産地消・農業体験を価値として求める消費者層の増加 | ✅ 農業ブランドの価値向上で働き手の関心を引きやすい |

### Technological（技術）

| 要因 | 内容 | 事業への影響 |
|------|------|-------------|
| スマートフォン普及率の高止まり | 農業マッチングアプリのインフラが既に整備済み | ✅ アプリ開発・展開コストが低い |
| 農業DX構想2.0（農林水産省、2024年） | 食と農全体のDX化が国策。2031年までにスマート農業技術活用割合50%以上が目標 | ✅ デジタルサービスへの農家の受容度が高まる |

| 電子契約・電子決済の普及 | 農業でも請求・支払いのデジタル化が可能に | ✅ 給与即日払い・電子契約でワーカーの利便性が向上 |

## 2. SWOT分析

> ※ PEST分析の機会・脅威をSWOTの外部要因に反映。

|  | **機会（Opportunity）** | **脅威（Threat）** |
|--|--|--|
| **強み（Strength）** | **SO（攻勢戦略）** 高齢農家向け出張サポートで、競合が取れない65歳以上農家層（全体70%）に先行獲得。政策補助金を活用しオンボーディングコストを下げる | **ST（多角化戦略）** 農家の採用予算縮小リスクに対し、完全成功報酬モデルを導入。採用できなければゼロコストの設計でdayworkの無料モデルに対抗 |
| **弱み（Weakness）** | **WO（弱点克服戦略）** ブランド認知がゼロの後発だが、Z世代・副業希望者向けのSNS施策とJA外の農業団体（直売所・農業法人組合）との連携でワーカー獲得コストを下げる | **WT（防御・撤退戦略）** 農家・ワーカーの鶏と卵問題は特定地域（1〜2県）に集中展開し、両サイドの密度を先に高めてから拡大する |

### 詳細

| 項目 | 内容 |
|------|------|
| **S：強み** | ①高齢農家向けUX・電話サポート（競合が手薄な層へのアプローチ）<br>②短期スポット→継続雇用への移行導線（LTV向上・競合との機能差別化）<br>③外国人労働者（特定技能）対応の多言語マッチング（競合ゼロの領域） |
| **W：弱み** | ①ブランド認知度ゼロ（後発参入）<br>②JA公認のdayworkとの信頼格差<br>③農家・ワーカー双方を同時に獲得するまでの初期マッチング密度問題 |
| **O：機会** | ①農業DX政策・補助金でデジタルサービス受容度が向上（PEST-P/T）<br>②副業解禁・Z世代の農業関心でワーカー供給が増加（PEST-S）<br>③地域計画の義務化で行政連携の窓口が広がる（PEST-P）<br>④地方移住トレンドで農業体験ニーズが継続的に増加（PEST-S） |
| **T：脅威** | ①dayworkが完全無料・JA連携で先行しており参入障壁が高い（競合）<br>②農業コスト高騰で農家の採用予算が縮小する可能性（PEST-E）<br>③農家の高齢化がデジタルサービス普及を遅らせるリスク（PEST-S）<br>④スマート農業化の進行で長期的には人手ニーズが縮小する可能性（PEST-T） |

## 3. 3C分析

### Customer（顧客）

#### 農家・農業法人（需要側）

**課題とニーズ**

| 項目 | 内容 |
|------|------|
| 最大の悩み | 繁忙期（収穫・田植えなど）の短期人材確保を毎回ゼロから行う手間とコスト |
| 規模感 | 農業経営体約104万経営体、うち農業法人3.1万社 |
| 採用の厳しさ | 農業経営体の半数近くが「以前より採用が難しい」と回答（農水省調査） |
| 繁閑の激しさ | 作業の進行状況・天候次第で必要人数が直前に変わる → 柔軟に求人変更できるツールが必要 |
| デジタルリテラシー | 40〜50代農家はアプリを活発利用。65歳以上はスマホ操作に抵抗感あり → 2層に分けた対応が必要 |
| 求める価値 | 採用手間の削減、コスト予測可能性、未経験者でも即日働ける仕組み（マニュアル整備）、定着率の向上 |

#### 副業希望者・短期就労希望者（供給側）

**特性とニーズ**

| 項目 | 内容 |
|------|------|
| 主要層 | 10〜40代（Z世代・ミレニアル世代）が8割以上 |
| 主な利用目的 | 副業（最多）→ パート・アルバイト→ 農業体験 |
| 動機 | 収入補完・食への関心（35.3%）・農業体験・地方暮らし志向 |
| 行動特性 | 朝型が多い（男性6:00〜、女性8:00〜スタートを好む） |
| 重視する点 | 時給だけでなく、職場の雰囲気・人間関係・作業のわかりやすさを重視 |
| 潜在市場 | Z世代の65%が農業に関心。地方移住相談件数は4年連続過去最高（2024年61,720件） |

### Competitor（競合）

| 比較項目 | daywork | 農How（アグリトリオ） | シェアグリ | **自社（想定）** |
|----------|---------|----------------------|-----------|----------------|
| サービス形態 | 1日単位マッチング | 1日単位スポット求人 | 超短期求人マッチング | スポット〜継続雇用移行型 |
| 料金モデル | **完全無料** | 300円/時間（農家負担） | 成功報酬型 | 成功報酬＋継続雇用月額 |
| 農家登録数 | 全国（JA連携） | 約850件 | 非公開 | — |
| ワーカー数 | 年間**18万人** | 約1.2万人 | 非公開 | — |
| マッチング率 | **83%**（2023年実績） | 非公開 | 非公開 | — |
| 連携機関 | **JA全国**（最大の強み） | 武蔵精密工業（親会社） | 地方自治体・財団 | JAを介さない農業法人・直売所 |
| 外国人対応 | ❌ | ❌ | ❌ | **✅（差別化ポイント）** |
| 継続雇用移行 | ❌ | ❌ | ❌ | **✅（差別化ポイント）** |
| 高齢農家サポート | ❌（弱み） | ❌ | ❌ | **✅（差別化ポイント）** |
| 主な弱み | 収益モデル不明瞭・高齢農家のアクティブ率低い | 地域カバレッジ限定・割高感 | 規模小・体験寄りで実務ニーズとズレ | 後発・認知度ゼロ |

**競合地図の示唆**

- 3社すべてが「スポット・1日単位」に閉じており、**継続雇用・外国人労働者・高齢農家サポート**の3軸が完全に手つかず。
- dayworkの無料モデルに真正面から価格競争を挑むのは不利。「スポット完結型」との棲み分けを図り、継続雇用ニーズに特化するレイヤーで優位に立つのが現実的。

### Company（自社）

**サービスの定義**

農業従事者と短期・副業・スポット就労希望者をマッチングし、農業現場の人手不足を解消する。スポット採用をきっかけに、農家とワーカーの継続雇用関係へスムーズに移行できる導線を持つことが最大の特徴。

**現状のポジション（仮説）**

| 項目 | 内容 |
|------|------|
| 提供価値 | ①繁忙期の即時マッチング ②スポット→継続雇用への移行 ③高齢農家向け対面サポート ④特定技能外国人の多言語マッチング |
| 収益モデル（案） | マッチング成立時の成功報酬（農家負担）＋継続雇用契約移行後の月額利用料 |
| 初期戦略 | 特定地域（1〜2県）への集中展開 → 両サイドの密度を高めてからエリア拡大 |
| 競合優位の源泉 | dayworkが取れていない高齢農家層（65歳以上、農家全体の70%）と継続雇用ニーズ |
| 克服すべき課題 | ①後発ゆえのブランド認知ゼロ ②農家・ワーカー双方を同時獲得するまでの鶏と卵問題 ③JA連携なしでの農家獲得コストの高さ |
