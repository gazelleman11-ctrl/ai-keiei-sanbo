# 事業検証レポート：農業従事者向けマッチングアプリ

**作成日：** 2026年6月25日

---

## 1. 課題仮説の検証

### この課題は実際に多くの人が抱えているか

農業分野の人手不足は、データで裏付けられた深刻かつ構造的な課題である。

| 指標 | 数値 |
|------|------|
| 農業・林業就業者数（2024年） | **180万人**（前年比7万人減） |
| 基幹的農業従事者（2023年） | **111万4千人**（2000年比▲54%） |
| 65歳以上の割合 | **約70%** |
| 49歳以下の割合 | **約10%** |
| 平均年齢 | **69.2歳** |
| 新規就農者数（2021年） | **5万2,700人**（2015年の6万5,000人から減少） |
| 年間農業離職者 | **約2万7,000人**（所得不足・労働条件が主因） |

### 課題を裏付けるデータや事例

- **悪循環の構造**: 高齢化 → 引退 → 後継者不在 → 耕作放棄地拡大 → さらなる人手不足、という負のスパイラルが固定化している。
- **繁閑差の問題**: 農業は収穫期に集中的な人手が必要だが、通年雇用が困難なため、既存の人材紹介・派遣サービスでは対応しづらい。
- **IT 活用の遅れ**: デジタルマッチングサービスが既に複数存在するものの、登録しても活用できない高齢農家が多く（年齢層が高いほど求人掲載率が低い）、需給のミスマッチが残っている。
- **政策的追い風**: 農林水産省は「雇用就農者の確保・育成」を重点施策として位置づけており、農業マッチングの行政連携・補助金活用が進んでいる。

**結論**: 課題は実在し、かつ拡大中。180万人の就業者減少トレンドが止まっていないことから、マッチング需要は今後も増加すると判断できる。

## 2. 市場規模の推定

### 国内市場規模（TAM・SAM・SOM）

農業専門マッチング市場の公式統計は存在しないため、就業者数・労働単価・マッチング手数料率を基に積み上げ推計を行った。

#### TAM（総獲得可能市場）
農業雇用労働全体をデジタルマッチング経由で代替した場合の市場。

> **農業就業者180万人** × **繁忙期平均30日稼働** × **日給8,000円** × **マッチング手数料率10%** ≈ **約430億円**

#### SAM（実際にアクセス可能な市場）
都市近郊農家・施設農業・農業法人など、デジタルサービスの普及が進む層（全体の20〜30%）。

> TAM × 20〜30% ≈ **約86〜130億円**

#### SOM（初期3年で現実的に獲得できる市場）
サービス立ち上げ後3年でのターゲット農家シェア（SAMの5〜10%）。

> SAM × 5〜10% ≈ **約4〜13億円**

### 市場の成長トレンド

- 人材紹介業界全体（2024年度）：**4,490億円**（前年比**+12%増**）— 全業種でデジタルマッチング移行が加速中。
- 農業従事者の減少ペース（年間約6〜7万人減）が続く限り、マッチング需要は**構造的に増加**。
- 農林水産省・地方自治体によるデジタル農業支援予算の拡充（スマート農業推進施策）が、プラットフォーム普及の追い風になっている。
- daywork の年間利用者が2025年時点で**18万人**に達しており、市場の実需が既に形成されていることを裏付けている。

## 3. 主要競合3社の調査

### ① daywork（デイワーク）

**運営**: JA（農業協同組合）と連携して開発・全国展開  
**サービス概要**: 1日単位で農家と働き手をつなぐスマートフォンアプリ。求人掲載から応募・採用まで完全アプリ内で完結。

| 項目 | 内容 |
|------|------|
| 料金 | **農家・求職者・JA いずれも完全無料** |
| 利用者規模 | 年間**18万人**（2025年）、サービス開始から6年で全国展開 |
| マッチング率 | **83%**（2023年実績：1.5万人募集 → 1.3万人成立） |

**強み**
- JAとの公的連携による農家側の高い信頼感
- 完全無料モデルで参入障壁が低く、利用者が急速に拡大
- 副業・週末利用層（10〜40代が8割以上）を取り込んでいる

**弱み**
- 収益モデルが不明瞭（完全無料のため持続的マネタイズが課題）
- 高齢農家のスマホ操作に対する抵抗感から、登録しても求人を出せないケースが多い
- 時給が相場より低めで、労働者側の定着率に課題
- 農業バイトのみに限定され、職域の広がりがない

---

### ② 農How（ノウハウ）

**運営**: 株式会社アグリトリオ（武蔵精密工業の新規事業）、愛知県豊橋市発  
**サービス概要**: 農家と個人をつなぐ1日単位のスポット求人サービス。登録から給与支払いまでスマホ完結。作業マニュアル動画による事前研修機能が特徴。

| 項目 | 内容 |
|------|------|
| 料金 | 農家：登録・掲載無料、**サービス利用料300円/時間**（ワーカー1人あたり） |
| 農家登録数 | 約**850件**（2023年9月時点） |
| ワーカー登録数 | **1万2,000人**超（2023年9月時点） |

**強み**
- 「農作業マニュアル動画」の事前共有機能で農業未経験者でも即戦力化
- 時間課金モデルで農家が費用を予測しやすい
- 製造業大手発の事業で資本基盤が安定

**弱み**
- 時間課金モデルは短時間利用の農家には割高感
- 規模がdayworkと比べると小さく、地域カバレッジが限定的
- 首都圏・関東エリアでの認知度が低い

---

### ③ シェアグリ

**運営**: 株式会社シェアグリ（ガイアックス社発・スピンアウト）  
**サービス概要**: 農業体験に興味を持つ人・副業希望者と農家を繋ぐ「超短期求人マッチング」。地域の自治体・財団との連携が特徴。

| 項目 | 内容 |
|------|------|
| 料金 | 農家：掲載無料、**成功報酬型手数料**（マッチング成立時に報酬額に応じた手数料） |
| 特徴 | 地方自治体・地域財団との連携（例：こゆ財団との協定） |

**強み**
- 成功報酬型で農家のリスクが低い
- 「農業体験」という体験価値訴求で、農業未経験の都市部ユーザーを集客しやすい
- 地域連携による公的サポートで信頼性が高い
- 新規就農から短期バイトまで多様な求人タイプに対応

**弱み**
- 利用者・農家数の規模が3社の中で最小
- 体験重視のポジショニングのため、生産性・即戦力性をシビアに求める農家ニーズとミスマッチが起こりやすい
- 収益規模・持続性が外部から見えにくい

## 4. 差別化ポイントの整理

競合3社はいずれも「スポット・1日単位の短期マッチング」に集中しており、以下の3軸で自社が優位に立てる余地がある。

### 差別化軸①：高齢農家でも使えるUX・サポート体制

**競合の共通の弱み**: dayworkは高齢農家のデジタルリテラシー不足でアクティブ率が低い。農How・シェアグリも同様。  
**自社の勝ち筋**: タブレット対応の大文字UI、電話サポート付きオンボーディング、JAや農業委員会と連携した「農家向け出張登録支援」を提供することで、競合がリーチできていない65歳以上農家層（全体の70%）を取り込む。

---

### 差別化軸②：継続雇用・長期マッチングへの対応

**競合の共通の弱み**: 全競合が「1日単位」のスポット特化であり、継続的な農業パートナー探しには不向き。  
**自社の勝ち筋**: 短期スポットから始まり、双方が希望すれば週次・月次の「継続雇用契約」に移行できるスムーズな導線を設計する。農家は毎回採用コストをかけずに固定ワーカーを確保でき、ワーカーは安定収入を得られる。マッチング単価・LTV（顧客生涯価値）が飛躍的に高まる。

---

### 差別化軸③：外国人労働者・特定技能人材との統合マッチング

**競合の共通の弱み**: 全競合のサービスが日本語話者の国内ワーカーを前提としており、農業が頼りにしている外国人労働者（技能実習・特定技能）には対応していない。  
**自社の勝ち筋**: 特定技能「農業」区分に対応した多言語マッチング機能（ベトナム語・インドネシア語）と、登録支援機関との提携による書類サポートをセットで提供する。農業の外国人就労者数は年々増加しており、競合が手つかずのブルーオーシャン市場を先行者として獲得できる。
